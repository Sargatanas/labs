5. Написать скрипт, который создает ссылку на страницу на вашем сервере. Например, если вы находитесь на www.mycoolsite.ru, JavaScript создаст ссылку на www.mycoolsite.ru/index.html.


Средствами javascript при просмотре из локальной папки невозможно решить данную задачу, т.к. предназначенное для этих целей обращение window.location.hostname не может считать корневое имя хоста на сервере.

Для демонстрации невозможности работы раскомментировать скрипт
